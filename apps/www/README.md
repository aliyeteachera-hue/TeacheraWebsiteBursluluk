# @teachera/www

Production host: `teachera.com.tr` (`www.teachera.com.tr`)

## Runtime ownership
- React/Vite web frontend
- Public pages + panel frontend routes
- No backend business logic (API served from dedicated API runtimes)

## Commands
- `npm --prefix apps/www run dev`
- `npm --prefix apps/www run build`

## Deploy
Deploy this runtime from `apps/www` with `vercel.json` in this folder.
