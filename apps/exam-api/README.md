# @teachera/exam-api

Production host: `exam-api.teachera.com.tr`

## Runtime ownership
- Candidate exam session APIs
- Forms API
- Exam health endpoint
- Uses shared backend modules from `packages/shared/backend`

## Commands
- `npm --prefix apps/exam-api run build`

## Deploy
Deploy this runtime from `apps/exam-api` with `vercel.json` in this folder.
