# @teachera/panel-api

Production host: `panel-api.teachera.com.tr`

## Runtime ownership
- Panel auth/session/password reset APIs
- Panel dashboard/candidates/notifications/dlq/settings APIs
- Panel health endpoint
- Uses shared backend modules from `packages/shared/backend`

## Commands
- `npm --prefix apps/panel-api run build`

## Deploy
Deploy this runtime from `apps/panel-api` with `vercel.json` in this folder.
